# Louresmolde Website Project

## Project Overview

This is a complete website for **Louresmolde - Construções Metálicas**, a Portuguese metalworking company specializing in metal constructions.

## Company Details

- **Company Name**: Louresmolde - Construções Metálicas
- **Industry**: Metalworking, Iron Forging, Metal Construction
- **Location**: Póvoa de Santa Iria, Forte da Casa, Portugal
- **Language**: Portuguese (pt)
- **Experience**: 20+ years in the industry

## Website Structure

### Main Pages
- **index.html** - Homepage with hero section, services overview, and company highlights
- **oqfazemos.html** - "What We Do" - detailed services page
- **catalogos.html** - Product catalogs and portfolios
- **obras.html** - Completed projects showcase
- **contactos.html** - Contact information and location
- **pedido_orcamento.html** - Quote request form

### Components & Architecture
- **components/Header.jsx** - Navigation header component
- **components/Footer.jsx** - Site footer component  
- **pages/Home.jsx** - Home page React component
- **App.js** - Main React application file
- **style.css** - Custom styling and overrides

## Services Offered

1. **Portões e Gradeamentos** - Gates and railings fabrication
2. **Escadas e Corrimões** - Iron stairs and handrails
3. **Peças Artesanais** - Handcrafted wrought iron pieces
4. **Estruturas Metálicas** - Metal structural projects
